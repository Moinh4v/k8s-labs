## Yaml 

Dictionary vs  List  vs List of dictonaries

1) Dictionary/ Maps 
```yml
Banana:
    Calories: 105
    Fat: 0.4g
    Carbs: 27g
```
2) List 
```yml
- Blue car 
- Green Car
- Yellow car
```

3) List of dictionaries
```yml
- Color: Blue
  Model:
     Name: Corvette
     Model: 1995
  Transmission:  Manual
  Price: $20,000
- Color: Green
  Model:
     Name: Corvette
     Model: 1995
  Transmission:  Manual
  Price: $20,000
- Color: Yellow
  Model:
     Name: Corvette
     Model: 1995
  Transmission:  Manual
  Price: $20,000
```

Dictionary - Unordered
List/Array - Ordered

Any line beginning with an # is a comment
